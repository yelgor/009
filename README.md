# Структура проєкту (папки)

## `src/assets/`
- **Відповідає за:** статичні файли (зображення/іконки), які імпортуються у компоненти та сторінки.
- **Тут має зберігатись:** `.png`, `.svg`, `.webp`, іконки, логотипи, емблеми.
- **Вже є:** `logo-009.png`, `ucu-emblem.png`, `asf-emblem.png`.

## `src/styles/`
- **Відповідає за:** глобальні стилі та дизайн-токени, спільні для всього сайту.
- **Тут має зберігатись:** `globals.css` (база/reset), `tokens.css` (CSS-змінні: кольори, відступи, шрифти, радіуси).
- **Вже є:** `globals.css`, `tokens.css`.

## `src/components/`
- **Відповідає за:** перевикористовувані UI-блоки (не сторінки).
- **Тут має зберігатись:** компоненти типу `Navbar`, `Hero`, `Footer`, спільні кнопки/карточки/модалки тощо. Кожен компонент у своїй папці: `Component.jsx` + `Component.module.css`.
- **Вже є:** `Navbar/`, `Hero/`, `Footer/` (з відповідними `.jsx` та `.module.css`).

## `src/pages/`
- **Відповідає за:** сторінки (роути) — те, що відкривається як окремий URL.
- **Тут має зберігатись:** папки сторінок `Home/`, `Equipment/`, `Account/`, `Docs/` тощо. Усередині: `Page.jsx` + `Page.module.css`. Сторінка збирає компоненти, але не містить перевикористовуваний дрібний UI (його виносимо в `components/`).
- **Вже є:** `Home/` (та інші сторінки, якщо вже додані командою).

## `src/ (корінь)`
- **Відповідає за:** запуск застосунку та підключення основних речей (роутинг/збірка).
- **Тут має зберігатись:** `App.*`, `main.*` або `index.*` (bootstrap React, router), за потреби `App.css`.
- **Вже є:** `App.*` та файл запуску (`main.*`/`index.*`) — залежно від шаблону (Vite/CRA).
